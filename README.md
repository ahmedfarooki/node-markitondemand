node-markitondemand
===================

Node.js library for accessing the http://dev.markitondemand.com APIs.

###Author
Ahmed Farooki - ahmedfarooki@gmail.com

###Running Tests
To run tests (requires mocha):

```bash
    $ npm test
    or
    $ mocha
```

```bash
To run test coverage (requires jscoverage):

    $ npm run-script coverage
    or 
    $ make coverage
```
